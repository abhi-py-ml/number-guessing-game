# Number Guessing Game

A clean and modular command-line number guessing game written in Python.

## Features
- Random number generation
- Input validation
- Option to quit mid-game
- Replay functionality
- Clean function-based architecture

## How to Run
python guessing_game.py

## Architecture
generate_number() – Generates random number
get_guess() – Handles validated input
check_guess() – Encapsulates game logic
main() – Controls program flow
